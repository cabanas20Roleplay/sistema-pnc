# 👮 SISTEMA DE INTELIGENCIA - PNC EL SALVADOR
### Proyecto Oficial: Cabañas 2.0 Roleplay

Este repositorio contiene el sistema de base de datos digital para la **Policía Nacional Civil** del servidor Cabañas 2.0 en ERLC.

## 🚀 Funcionalidades
* 🆔 **Portal Ciudadano:** Consulta de expedientes mediante login de Discord.
* 💰 **Economía en Tiempo Real:** Conexión directa entre el Bot de Discord y la Web.
* 🚨 **Antecedentes:** Registro de arrestos, multas y advertencias.
* 🪪 **Licencias:** Control de licencias de conducir y portación de armas.

## 🛠️ Tecnologías Usadas
* **HTML5 / CSS3:** Interfaz estilo terminal táctica.
* **JavaScript (ES6):** Manejo de OAuth2 de Discord.
* **Firebase:** Base de datos en tiempo real.
* **GitHub Pages:** Hosting del portal.
